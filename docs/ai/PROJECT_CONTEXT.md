# 3. Background

The Google WAXAL ASR Challenge aims to advance automatic speech recognition for African languages.

Participants are expected to develop robust speech recognition systems capable of generalizing across multiple speakers, recording conditions and languages.

Unlike many competitions focused only on leaderboard optimization, this challenge emphasizes **generalization**.

The final ranking is determined using a hidden evaluation dataset released only during Phase 2.

Consequently, building a robust and reusable ASR pipeline is more important than overfitting to the public leaderboard.
